## What
Measuring speed using (magnetic biltema bicycle speedometer) wheel rotation sensor.
* Displayed on 2x16 lcd
* Extra (debug) info avaiable via UART

## How
* FreeRTOS 
* IAR/Olimex LPC2138-SK
* EWARM 9.20.2
